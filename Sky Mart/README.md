# 201Mart

A minimal, dark-mode glassmorphic e-commerce web application designed for modern shopping experiences.

🌐 **Live Demo:** [Live Link](https://www.google.com/search?q=https://your-deployment-link.com)

---

## 🛍️ About The Project

**201Mart** is a full-featured e-commerce platform built using React, Tailwind CSS, and FakeStoreAPI. It combines clean glassmorphic aesthetic design with full storefront functionality—offering user authentication, an interactive catalog, detailed product views, and persistent cart management.

### Core Functionality

- **User Authentication:** Client-side registration and login workflows built with form validation and protected route access.
- **Dynamic Product Catalog:** Real-time search, category filters, and multi-parameter sorting by price and rating.
- **Product Discovery:** Detailed product pages equipped with category recommendations and sequential previous/next navigation.
- **Shopping Cart:** Persistent cart management powered by LocalStorage, featuring quantity steppers and dynamic order totals.
- **Modern Aesthetic:** Clean dark-mode interface styled with Tailwind CSS, custom iconography, and smooth physics-based scrolling.

---

## 🛠️ Tech Stack

- **Core:** React 19, React Router v7
- **Styling:** Tailwind CSS v4, Lucide Icons
- **State & Forms:** React Context API, React Hook Form
- **Data & Motion:** Axios, FakeStoreAPI, Lenis Scroll
